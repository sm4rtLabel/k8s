# WAO Test Setup

This project is set up for testing with WAO (Wizard AO SDK & Testing Framework).

## Prerequisites

1. **Install HyperBEAM**: Follow the instructions at [docs.wao.eco/hyperbeam/installing-hb-wao](https://docs.wao.eco/hyperbeam/installing-hb-wao)
2. **Node.js**: Make sure you have Node.js 18+ installed
